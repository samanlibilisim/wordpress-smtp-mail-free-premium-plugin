Configure SMTP settings for WordPress and send test emails directly from the WordPress dashboard.

== Description ==

Samanlı SMTP Mail allows you to send WordPress emails using your preferred SMTP server.

Features:

* SMTP Host
* SMTP Port
* SMTP Username
* SMTP Password
* SSL / TLS Encryption
* Sender Email
* Sender Name
* Test Email Tool
* Compatible with wp_mail()

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory.
2. Activate the plugin.
3. Navigate to Settings → SMTP Mail.
4. Configure your SMTP settings.
5. Send a test email.
